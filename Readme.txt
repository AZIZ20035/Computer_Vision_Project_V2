**Instructions to run the code**

1 - Data collection:

      run the datacollector.py python file to use device's camera to capture infinite images for training 
      and storing them into folder.

2 - Data preprocessing & Model training and testing

      run the cnn.py python file to load images for shuffling and splitting data ,train and test model and store it.

3 - Deployment

      run the AITrainer.py python file to finally use the application and get your answer.
